PRIVACY ENHANCED

As we know in most blockchain systems, all transaction records are available on public domain. NexBank ensures all transactions are anonymous as it deploys ring signature algorithm, the same currently used by Monero Blockchain

CENSORSHIP PROOF

NexBank has built-in support for OBSF4, obfuscation protocol that hides the nature of the traffic that goes through it. By incorporating OBFS4, it is possible to use NexBank even in countries where government attempts to restrict or block blockchain access, like China or Iran.

ENCRYPTED COMMUNICATION

NexBank takes the privacy of users very seriously. All chats or conversations are encrypted so no one cannot track or review any code. Symmetric encryption and Asymmetric encryption are utilized to ensure this objective is met.

ANYSMARTPHONE ANYTIME

Anyone with any smartphone (Android, IOS) can download NexBank app and start using it anytime anywhere 24/7.

HACK PROOF

Blockchain stores information across a number of computers. Because information is scattered over a vast number of locations, this renders the network decentralised and hence makes hacking NexBank near impossible. To further enhance this feature, NexBank’s adopts 4 tiers security; biometric scan, User track Check (UTC) System, Server-Side Validation, Intrusion Defense System (IDS) and Intrusion Prevention System (IPS).
